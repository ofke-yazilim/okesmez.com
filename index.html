Hakkımda

Üniversiteden mezun olduğum 2012 yılından itibaren gerek bireysel gerekse kurumsal firmalarda web projeleri kodlayarak çalışma hayatımı sürdürdüm. Daha önce çalışmış olduğum firmalarda PHP ya da JAVA programlama dillerini kullanarak b2b, b2c eticaret sistemleri ya da içerik yönetim sistemleri(CMS) geliştirdim. Laravel Frameworkunu kullanarak belediye web portalları kodladım. Yazmış olduğum projelerde jQuery ve Bootstrap kütüphanelerini, Memcache, Elasticsearch, Redis ve MongoDb NoSQL yapılarını aktif olarak kullanmaktayım. Mobil uygulamalar için Rest ve Soap web servisler geliştirdim. Halen kurumsal bir firmada muhasebe destekli eticaret sistemleri ve belediye web portalları geliştirmekteyim. Kullandığım Programlama Dilleri ve Teknolojiler : PHP 7x, PHP 5.6, Laravel 5x, JAVA, jsp, ,Jquery kütüphaneleri, Bootstrap, CSS, MySql, MsSql, Smarty Template Engine, Rest Api, Soap, Memcache, Elasticsearch, Redis, MongoDb vb.
